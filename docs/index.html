软件源安装

octave_tar可以通过Octave软件源安装。

通过Octave软件源安装octave_tar，代码如下：

>> pkg install -forge octave_tar

在线安装

octave_tar可以在线安装。

在线安装octave_tar，代码如下：

>> pkg install 'https://github.com/CNOCTAVE/octave_tar/releases/download/1.0.1/octave_tar.tar.gz'

源码安装

octave_tar可以直接使用tar包安装。

假设你下载的源码包名为octave_tar.tar.gz，源码安装的代码如下：

>> pkg install octave_tar.tar.gz -local

如何使用
在安装后，加载octave_tar即可使用，代码如下：

>> pkg load octave_tar

依赖库
无
函数用法
tar_pack

tar_pack (SOURCE1, SOURCE2, ... , OUTPUT_FILENAME)

输入一个或多个文件或文件夹名SOURCE1, SOURCE2, ...和打包后的文件名OUTPUT_FILENAME。

如果返回0，则代表打包成功；否则代表打包失败。

tar_unpack

tar_unpack (TAR_FILENAME, DESTINATION_DIR)

输入打包文件名TAR_FILENAME和解包后的文件名DESTINATION_DIR。

如果返回0，则代表解包成功；否则代表解包失败。

常见用例
打包一个名为file的文件为file.tar，代码如下：

>> tar_pack("file", "file.tar")

打包一个名为dir的文件夹为dir.tar，代码如下：

>> tar_pack("dir", "dir.tar")

打包一个名为file的文件和一个名为dir的文件夹为new.tar，代码如下：

>> tar_pack("file", "dir", "new.tar")

解包一个名为file.tar的文件至new_file文件夹，代码如下：

>> tar_unpack("file.tar", "new_file")

解包一个名为file.tar的文件至当前文件夹，代码如下：

>> tar_unpack("file.tar", ".")
